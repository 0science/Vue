# 尚硅谷Vue2学习

## 创建项目

### 使用脚手架创建

```
# 安装 vue-cli
npm install -g @vue/cli

# 安装错误时,强制安装
npm install -g @vue/cli --force

# 检查 vue-cli 版本
vue -V


```


### 使用Vite创建

```
npm create vite my-template -- --template vue
```

### 项目结构

```
src/

│   ├── HomePage.vue
│   ├── AboutPage.vue

```
